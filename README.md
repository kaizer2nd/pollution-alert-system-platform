# Pollution Alert System Platform

Frontend-only, presentation-ready demo built with React + Vite, Tailwind CSS, Framer Motion, Recharts, Swiper, and Lucide icons. All data is mocked locally—no backend or APIs.

## Folder Structure

- `client` - React + Vite frontend
- `server` and `uploads` - Legacy folders (not used by this frontend demo)

## How to Run

1. Install frontend dependencies:
   ```
   cd client
   npm install
   ```
2. Start the dev server:
   ```
   npm run dev
   ```
3. Open the app:
   - `http://localhost:5173/`
